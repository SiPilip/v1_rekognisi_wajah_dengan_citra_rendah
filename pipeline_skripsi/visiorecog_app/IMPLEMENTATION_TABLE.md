### Tabel Implementasi Kelas

| Nama Kelas | File Sumber Utama | Deskripsi Implementasi |
| :--- | :--- | :--- |
| **`FrontendUI`** | `frontend/src/App.tsx` | Direpresentasikan oleh komponen `App` dalam React, yang berfungsi sebagai kelas utama di sisi frontend. Dibangun menggunakan TypeScript dengan pustaka Material-UI untuk komponen antarmuka dan Axios untuk melakukan komunikasi HTTP asinkron dengan backend. Kelas ini mengelola state aplikasi, menangani interaksi pengguna, dan menampilkan data hasil analisis. |
| **`FastAPI_App`** | `backend/app/main.py` | Merupakan instance dari aplikasi FastAPI yang berfungsi sebagai web server. Kelas ini mendefinisikan semua endpoint REST API (seperti `/recognize`), mengonfigurasi CORS untuk komunikasi dengan frontend, serta menginisialisasi dan menyediakan akses ke `FaceRecognitionPipeline`. |
| **`FaceRecognitionPipeline`**| `backend/app/pipeline.py` | Kelas inti di backend yang berisi seluruh logika pemrosesan. Kelas ini mengintegrasikan berbagai pustaka pihak ketiga: **GFPGAN** untuk restorasi wajah, **DeepFace** (model ArcFace) untuk ekstraksi fitur, **PyIQA** untuk penilaian kualitas gambar, dan **Scikit-learn** untuk memuat dan menggunakan model klasifikasi. |
| **`Classifier` (Interface)** | `backend/app/pipeline.py` | Sebuah konsep desain (*interface*) yang diimplementasikan oleh model-model klasifikasi. Dalam kode, ini diwujudkan melalui pemanggilan metode `predict_proba` yang seragam pada model KNN dan SVM yang telah dimuat dari file `.pkl`. |
| **`KNN_Model` & `SVM_Model`**| `backend/models/*.pkl` | Model `KNeighborsClassifier` dan `SVC` dari Scikit-learn yang telah dilatih dan disimpan sebagai file `.pkl`. Model-model ini dimuat oleh `FaceRecognitionPipeline` saat startup untuk melakukan prediksi identitas berdasarkan fitur wajah yang diekstraksi. |
| **`CosineSimilarity`** | `backend/app/pipeline.py` | Diimplementasikan sebagai sebuah metode di dalam `FaceRecognitionPipeline`, bukan sebagai kelas terpisah. Menggunakan fungsi `cosine` dari pustaka `scipy` untuk menghitung jarak kosinus antara embedding gambar masukan dengan semua embedding di galeri, yang berfungsi sebagai metode pencocokan 1:N. |
| **`AnalysisResult` & `PipelineResult`**| `frontend/src/App.tsx` | Struktur data yang digunakan untuk transfer informasi antara backend dan frontend. Di backend, ini adalah *dictionary* Python yang diserialisasi menjadi JSON. Di frontend, strukturnya didefinisikan secara ketat menggunakan *interface* TypeScript untuk memastikan keamanan tipe data saat menerima respons dari API. |
